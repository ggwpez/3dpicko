Introduction

About

This is the manual belonging to the colony picker C3PickO. The "User Manual" is directed to the common user of the picker and explains how to use it and the software. "Configuration" describes how to configurate the hardware and software. Hence it targets rather administrators, but it explains tasks a capable user can handle as well. "Administration" is targeting administrator only and explains how to setup the hardware and software.

The "User Manual" consists of three chapters. "First Steps" explains what precondition have to be met for operating the device. "Example Picking Process" demonstrates how a successful colony picking process is done. "Advanced Settings" all further setting are explained.

Please note "that" the specification given are "tailored to and testet with the hardware provided by the "TU Darmstadt" (camera 3D printer, 3d printed parts). Howerver they can be transfered to similar hardware. The contact person is Prof. Dr. Johannes Kabisch.

First Steps

Before starting any picking process please make sure all preconditions are met. These steps are very important. Not complying with them can notonly destroy your experiment but also damage the device.

Check if the following is given:
  • The colony picker is located on a plane non-flammable surface in a well ventilated room.
  • There is enough filament for processing the colony pick (for each colony about 1 cm filament is needed).
  • The filament has to be extruded up to the end of the nozzle.
  • There is enough space in the trashcan to accomidate the waste.
  • There must be air pressure available, you can test this with manually activating the pneumatic scissors.
  • The picker must be connected to power and the red power button on the right has to be switched on (the display is no indication since it can also be lit when the picker is turned off).
  • The picker needs a working network connection.

Advanced Settings

Choose Colonies

Detection Settings

Algorithm

Here you can select the Algorithm which will be used for the colony detection. There are two choices either "Normal" or "Fluro". Normal is for any kind of colonies while Fluro is only intended for fluorescent.

Area

With this setting you can select what size the colonies you want to pick should possess, you can select a lower as well as an upper limit. In the example image you could filter out any of the three circles alone or the middle sized one and one of the others.

Safety Margin

This setting is used to exclude all colonies which are to close to the Petri dish wall to be picked. This is a saftey feature to ensure that the head does not collide with any wall.

AABB Side Ratio

This option lets you choose what ratio of length and width the colony can have. A perfectly round colony would have the ration 1. In the example image refer to the green bounding box, the calculation is as follows: whatever side is shorter is divided by the longer side.

Calculation:
The shorter side of the bounding box is taken and divided by the longer side.

BB Side Ratio

Here one can choose the ratio of length and hight of the colonies to be included. For this a bounding box is laid around the colony and rotated to be as small as possible. Refer to the red bounding box.

Calculation:
The shorter side of the rotated bounding box is taken and divided by the longer side.

Convexity

This is option filters the convexity, it processes the outline of the colony and checks if the arc is evenly, the more evenly it is the higher the number becomes.

Calculation:
First the convex hull of the colony is approximated. Afterwards the area of the recognized hull is divided by the area of the approximated hull.

Circularity

This option checks the roundness of the colonies The rounder the colony the higher the number.

Calculation:
This calculates the perfect circumference2 of the colonies area and divides it by the recognized circumference2.

Choose Plates

Number of Colonies

Here is defined how many colonies are being picked. If less colonies are selected than recognized the excess is discarded. The selection of colonies is randomized.

Configuration - Hardware

Refilling the Filament

For replacing the filament operate the handle at the stepper motor.

Now the stepper motor lost grip and the present filament can be pulled out. Afterwards the new filament can be inserted in the handles hole and pushed past the stepper motor into the tube. Push the filament until it reaches the nozzle, sometimes it tangles with the brass tube connection. To fix this the tube connection can be unscrewed and the filament manually placed into the brass adapter and screwed on again. When finished make sure the filament is located at the tip of the nozzle. The filament needs to be steril, if there are any contaminations, they can be transmitted to the hardware and recontaminated a future filament.

Only if this is met the printer can determine the filament exact location and pick correctly.

Testing Pneumatic Scissors

Manually press the switch right sided to the scissors to activate them. They should close automatically and open after releasing the switch.

Administration - Hardware

For the commissioning of the device following steps should be performed. It is presumed that the printer is already transformed into a colony picker.

Overview

The colony picker consist of a 3d printer which is controlled by a mini pc (Raspberry Pi). The pc is getting its commands from a server located elsewhere. The heated bed as well as the heating element for the nozzle are disconnected and should not be operable. The colony picker needs a mains power source, a network connection and 5 bar air pressure via a 6mm quick coupling.

Step by Step Initial Step

This step by Step guide will bring you through the assembly of the colony picker, it is assumed that all parts are already on hand.

1. Assembly

Assemble the printer according to the manufactures instruction. But do not mount or connect the heating bed. Now screw the pneumatic scissors into the Petri dish holder. Now the Petri dish holder can be mounted onto the printer to do this first place the screws heads up inside the frame and fixated it with a nut (This nut will now hold the Petri dish bed onto the heating element). A further nut has to be placed on top of the last on, for later high adjustment. Afterwards put the screws through the dedicated holes of 3d printer's heating bed holder and fixated it with another nut. Now assemble the switch for the pneumatic scissors to do this glue it into the switch holder and position it into the Petri dish holder.
Heating-bed holder:

switch holder with attached switch:

2. Positioning

Position the colony picker on a plane surface, make sure it has enough space to all sides. Also check the 3d printer’s manual for further instruction.

3. Wiring

Raspberry Pi

Connect the raspberry pi to the printer with mini usb cable and with a network cable to the network. Lastly connect it to the printers power supply.
3D printers USB connection:

Pneumatic Scissors

The pneumatic scissors have to be connected to the magnetic valve. Please check the image for the correct orientation.

Magnetic Valve

The magnetic valve has to be connected to the contact switch mounted on the Petri dish holder and to air pressure check the attached images for the correct orientation.

3D Printer

Finally plug the 3d printer into the main power-line, as soon as you turn it on, also the attached Raspberry Pi will start.

4. Calibration

It is important, that the socket is leveled, otherwise the ilament might not puncture all colonies or might even collide with a Petri dish. For leveling it slowly move the printers head to the front left corner of the Petri dish holder (e.g. by hand or webinterface of the pickers mini pc). After you reached that location slowly move it to the back and adjust the nuts, so that the distance between nozzle and Petri dish holder stay the same. After you reached the back do the same moving right.

5. Filament

Insert the filament, check "Refilling the Filament" in the chapter "Configuration Hardware" for this step.

Administration - Software

Install Software

There are two options available for installing the software, either use the "Ready to use VM" or the "Manual Installation

Ready to use VM

The ready to use software is delivered as a hdd image and an XML configuration file for the vm. This image has to be installed on a server. The network configuration has to be set up so that the server is accessible by the users and the server has access to the mini pc controlling the picker. The user's pc does not necessarily need a direct connection to the printer, however for trouble shooting purposes it is recommended (e.g. for manual controlling the picker).

Manual Installation

The Software needs some libraries e.g. for the image recognition. For installing all the dependencies as well as the software there is a script available which will do that for you. Natural you can use the script to check those commands.

Running the Software

The configuration for the web server is saved in the "serverconfig.ini" it is located in 3cpicko/GUI/. By default it is in working condition but for adding encryption or changing the standard port (8080) it has to be modified. For running the service start the "Main" program and give it as parameter the "serverconfig.ini".
