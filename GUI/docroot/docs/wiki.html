Introduction

About

This is the manual belonging to the colony picker C3PickO. The "User Manual" is directed to the common user of the picker and explains how to use it and the software. "Administration" is targeting administrator only and explains how to setup the hardware and software and how to configure the picker.

The "User Manual" consists of three chapters. "First Steps" explains what precondition have to be met for operating the device. "Example Picking Process" demonstrates how a successful colony picking process is done. "Advanced Settings" explains all further settings.

Please note that the specification given are geared to and testet with the hardware provided by the TU Darmstadt (camera 3D printer, 3d printed parts). However they can be transfered to similar hardware. The contact person is Prof. Dr. Johannes Kabisch.

First Steps

Before starting any picking process please make sure all preconditions are met. These steps are very important. Not complying with them can not only destroy your experiment but also damage the device.

Check if the following is given:
  • The colony picker is located on a plane non-flammable surface in a well ventilated room.
  • There is enough filament for processing the colony pick (for each colony about 1 cm filament is needed).
  • The filament has to be extruded up to the end of the nozzle.
  • There is enough space in the trashcan to accomidate the waste.
  • There must be air pressure available, you can test this with manually activating the pneumatic scissors.
  • The picker must be connected to power and the red power button on the right has to be switched on (the display is no indication since it can also be lit when the picker is turned off).
  • The picker needs a working network connection.
  • Make sure that you know where the printer's power button is. In case of any malfunctioning of the printer turn it off immediatly.

Administration - Configure Picker

Under the settings tab you must create at least one profile of each of the following types. With each profile you can adapt picking process to your demand to a certain extend.

Printer Profile

Profile Identifier:
An arbitrary name for the profile.
Movement speed:
The speed at which the printer moves on the x and y axis in mm/min. Consult your printer's datasheet. The maximum speed should be feasible.
Cut position:
The x,y position the colony picker moves to for cutting the filament. The z value must be high enough so that the nozzle of the printer does not hit the trigger of the scissors yet. x,y and z are given as absolute values in mm.
Push trigger at:
The height at which the cutting mechanism is triggered, absolute value in mm.
Filament length cut to source:
Defines how long the filament should stick out while the nozzle moves from the Cut position to the source plate, absolute value in mm.
Filament length source to master:
Defines how long the filament should stick out while the nozzle moves from the source to the master plate, absolute value in mm.
Filament length master to goal:
Defines how long the filament should stick out while the nozzle moves from the master to the target plate, absolute value in mm..
Filament length goal to cut:
Defines how long the filament should stick out while the nozzle moves from the target plate to the Cut position, absolute value in mm.
Filament length pick from source:
Defines how much the filament should stick out for for picking a colony on the source plate, absolute value in mm.
Filament length put onto master:
Defines how much the filament sticks out for putting a picked colony onto the master plate, absolute value in mm.
Filament length after cut:
Defines the length of the remaining filament after it was cut, absolute value in mm. This is roughly the distance between the height of the tip of the nozzle when it triggers the scissor and gap between the blades of the scissor. You can figure it out by moving the nozzle to the "cut filament position", extruding the filament as far as you want, ensuring its tip can be cut, and triggering the scissor. Then measure the length of the still sticking out filament and the lenght of the cut filament for the "length of removed filament" setting.
Filament length to be cut:
Defines how much filament should be cut off by the scissors, absolute value in mm.
Safety distance between nozzle and plates:
It is crucial that the colonies pickers head does not touch the walls of any Petri dish, however moving in z axis is slow. For this setting find a balance between safety and speed. Administration - Hardware

For the commissioning of the device following steps should be performed.

Parts

The colony picker consist of a 3d printer which is controlled by a mini pc (Raspberry Pi). The pc is getting its commands from a server located elsewhere. The heated bed as well as the heating element for the nozzle are disconnected and should not be operable. The colony picker needs a mains power source, a network connection and 5 bar air pressure via a 6mm quick coupling.

Here are additional items listed needed to assemble the device.

3D Printer

Raspberry Pi

Power Supply Unit

Magnetic Valve

Step by Step Assembly

These instructions will guide you through the assembly of the colony picker. It is assumed that all parts are already on hand.

1. Assembly of the printer

Assemble the printer according to the manufacture's instructions. Note that you do not need to mount or connect the heating bed. Insert a screw in the drilling at the scissors mounting place. Then mount the pneumatic scissors onto the socket. Now mount the socket onto the printer's bed holder. Fixate each screw with a nut (This nut will now hold the Petri dish bed onto the heating element). A further nut has to be placed on top of the last on for later leveling of the socket. Afterwards put the screws through the dedicated holes of the 3D printer's heating bed holder and fixate them with nuts again. At last attach the pneumatic scissor's switch to the socket. You can use tape or glue to fasten it.
Heating-bed holder:

switch holder with attached switch:

2. Positioning

Position the colony picker on a plane surface. Make sure it has enough space to all sides, so that the socket can move unhindered on the x,y axes. Also check the 3d printer’s manual for further instruction.

3. Wiring

Raspberry Pi

Connect the raspberry pi to the printer with mini usb cable and with a network cable to the network. Lastly connect it to the printers power supply.
3D printers USB connection:

Pneumatic Scissors

The pneumatic scissors have to be connected to the magnetic valve. Please check the image for the correct orientation.

Magnetic Valve

The magnetic valve has to be connected to the contact switch mounted on the socket. For checking air pressure manually press the switch right sided to the scissors to activate them. They should close automatically and open after releasing the switch.

3D Printer

Finally plug the 3d printer into the main power line. As soon as you turn it on, also the attached Raspberry Pi powers up.

4. Calibration

It is important that the socket is leveled. Otherwise the filament might not pick all colonies or might even collide with one of the plates inside the socket. For leveling it move the printers head carefully from one of the socket's corner to the other (e.g. by hand or webinterface of the pickers mini pc). While doing so adjust the nuts to the effect that the distance between nozzle and socket is everywhere the same.

5. Filament

Insert the filament. Operate the handle at the stepper motor.

Now the stepper motor lost grip and the present filament can be pulled out. Afterwards the new filament can be inserted in the handles hole and pushed past the stepper motor into the tube. Push the filament until it reaches the nozzle, sometimes it tangles with the brass tube connection. To fix this the tube connection can be unscrewed and the filament manually placed into the brass adapter and screwed on again. When finished make sure the filament is located at the tip of the nozzle. The filament needs to be steril, if there are any contaminations, they can be transmitted to the hardware and recontaminated a future filament.

Only if this is met the printer can determine the filament exact location and pick correctly.

Administration - Software

Install Software

There are two options available for installing the software. Either use the "Ready to use VM" or the "Manual Installation

Ready to use VM

The ready to use software is delivered as a hdd image and an XML configuration file for the vm. This image has to be installed on a server. The network configuration has to be set up so that the server is accessible by the users and the server has access to the mini pc controlling the picker. The user's pc does not necessarily need a direct connection to the printer, however it is recommended for trouble shooting purposes (e.g. for manual controlling the picker).

Manual Installation

The Software needs some libraries e.g. for the image recognition. For installing all the dependencies as well as the software you can use the provided script.

Running the Software

The configuration for the web server is saved in the "serverconfig.ini". It is located in 3cpicko/GUI/. It works out of the box. Modify it if you want to add encryption or change the standard port (8080). For running the service start the "Main" program and pass the "serverconfig.ini" as the parameter.
