Guide through the picking process

This guide will bring you through the picking process, you should already have done all preparation in "First Steps". Since this is intended to make the start into the picking process as easy as possible most functions are not mentioned here. To get a better understanding for those please refer to "Advanced Settings".

1. Taking the picture

This step has great impact on how easy and exact the picket process will be. Therefore this step should not be rushed and done with great diligence. Select the following camera settings:
aspect ratio: 16:9,
brightness: 0,
Iso: 100.
Now insert the Petri dishes you wish to have picked into the camera, the angled sides should be on the left hand side. Make sure the camera rotation is correct. It is possible to fade in a grid for getting the alignment between petri dish and edges of the picture right.

Now turn on the under light and zoom in as far as possible yet still covering the entire Petri dish. After you have pressed the camera shutter check if the picture has good contrast and take a new one if necessary. After you are satisfied with the result copy the picture to the pc you intend to start the picking process with. Here is an example of a good image:

2. Inserting the Petri dishes into the colony picker

Insert the Petri dishes into the colony picker, the orientation and placement are written into the frame. It is crucial that you comply with it, otherwise the picking process will fail and the colony picker can get damaged. It should look like this: Note the correct orientation!

Now enter the following URL into your browser: c3picko.bio.tu-darmstadt.de you will be welcomed with the following page:

4. Choose Image

You have two option to upload the picture you can either click on "Drop Image" and select the picture with your file explorer or drag and drop it in the dedicated area. If you have uploaded the image beforehand you can select it below the photo upload within "All Images".

5. Cut Image

In this step you have to cut the picture to the right size in order for the image recognition to be able to recognize the colonies and also to get the right offsets from all Petri dish walls for the picker (you don't want it to pick beside the colonies). For this you use your mouse to move the rectangle to correct position of the Petri dish. You can adjust the size when grabbing the edges. After you have cropped the image to the right size, you can click on "Next Step". This is an example:

7. Detect Colonies

The colony detection runs automatically when you reach this step. After changing any settings the colony detection will automatically be redone. Besides changing the settings on the right hand side you can also klick colonies in the picture to get more information about them and exclude them from your selection. The excluded colonies are shown in red. For more information about the different filter settings check out the [Advanced Settings](User-Manual/User-Manual-‚Äê-Advanced-Settings).

8. Choose Plates

Here you have to select what kind of PCR dish you want to use, click the drop down menu and select the appropriate one. Afterwards you can select with which well you wish to start. This is useful if your PCR dish is already partially occupied. The wells which will be used are shown in green. If you have selected more colonies than there is space available on the pcr dish the excess colonies will be discarded. This is an example of a 96 well pcr dish:

9. Overview

This site shows you all the configuration you have done before and is intended as a mean to exclude any errors before starting the picking process

10. Start Picking

Here you can finally start the picking process.

11. Finishing

After the picking process is complete you can remove the Petri dishes from the colony picker and start your experiments. Please don't turn the printer off via the power switch, first the integrated pc needs to be turned off. Best leave this to the administrators ;)
