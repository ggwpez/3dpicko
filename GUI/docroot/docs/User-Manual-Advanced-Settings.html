Detection Settings

Algorithm

Here you can select the Algorithm which will be used for the colony detection. There are two choices either "Normal" or "Fluro". Normal is for any kind of colonies while Fluro is only for fluorescent.

Area

With this setting you can select what size the colonies you want to pick should possess, you can select a lower as well as an upper limit. In the example image you could filter out any of the three circles.

AABB Side Ratio

This option lets you choose what ratio of length and with the colony can have. A perfectly round colony would have the ration 1. In the example image refer to the green bounding box, the calculation is as follows: whatever side is shorter is divided by the longer side.

$`min(width, height) / max(width, height)`$ whereas _width_ and _height_ are side lengths of an Axix Aligned Bounding Box.

BB Side Ratio

This option is similar to the one above, the only difference is that the bounding box is rotated to be as small as possible and not aligned with the edges of the image. Refer to the red bounding box.

$`min(width, height) / max(width, height)`$ whereas _width_ and _height_ are side lengths of an Rotated Minimum Area Bounding Box.

Convexity

This is option filters the convexity, it processes the outline of the colony and checks if the arc is evenly, the more evenly it is the higher the number becomes.

$`area(contour) / area(hull)`$ whereas _contour_ is the polygon representing the edge and _hull_ is the an approximated convex hull polygon.

Circularity

This option checks the roundness of the colonies The rounder the colony the higher the number.

$`(4 * \pi * area) / {circumference(contour)}^2`$ whereas _area_ is the area of the Colony detected in pixel and _contour_ is the polygon representing the edge.

Filter Colonies

This setting is similar to the Area setting, but instead of setting the limits directly you define a deviation from the calculated average colony. You have three options to choose from:

Area

How much smaller or bigger the colonies can be from the average.

Brightness

This is for selecting the brightness. It is possible to choose colonies with different colours or fluorescent ones. Note that different colours can have the same brightness. You might want to change the colour of the lightning while photographing to improve results.

Circumference

How much smaller or bigger the circumference of colonies can be from the average, this setting is interesting if you have non-round colonies.

With mean you select the actual deviation for filtering, in this example image, the area of colony two is larger than the area of colony one but the circumference of colony one is actually greater than the one of colony two. It is possible to use either area or circumference to select the colonies intended for the pick.
